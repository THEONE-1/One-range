# One 安全靶场 - 前端项目

基于 Vue 3 + Ant Design Vue 构建的现代化前端界面。

## 技术栈

- Vue 3
- Ant Design Vue 4.x
- Vue Router 4
- Pinia
- Vite 5
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

## 构建

```bash
# 构建生产版本
npm run build

# 输出到 ../src/main/resources/static/dist
```

## 说明

- 前端运行在 3000 端口
- 后端API代理到 8000 端口
- 登录后会保存用户信息到 localStorage
